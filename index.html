Metado da primeira lei de OHM

O metado se refere a maneira de calcular e achar os dados que precisa
num calculo.
O metado sendo bem simples de fazer, quando for precisar achar o U(Tensão volts),
vai precisar multiplicar R(resistecia OHM) pelo I(correntes ampares)
os outros dois vão so divisão, sendo eles
se for achar o R(resistecia OHM), vai dividir U(Tensão volts) pelo I(correntes ampares)
o ultimo é achar o I(correntes ampares), que é U(Tensão volts) por R(resistecia OHM)
